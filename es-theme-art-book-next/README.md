# Art-Book-Next theme port for ArkOS based on the Nintendo Switch Theme for EmulationStation

Main menu View

![https://batocera.org/upgrades/themes/Art-Book-Next.jpg](https://batocera.org/upgrades/themes/Art-Book-Next.jpg)

After installation under UI Settings set Transition Style to Slide and Game Launch Transition to Fade to emulate the art book next theme perfectly.
